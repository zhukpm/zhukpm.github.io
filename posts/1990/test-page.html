How to setup a virtual environment using venv for Python projects? Easy! Just follow the steps below.

Setting up a virtual environment using venv

  1. Open your terminal.
  2. Navigate to your project directory.
  3. Run the command: python3 -m venv venv
  4. Activate the virtual environment:
  5. On macOS/Linux: source venv/bin/activate
  6. On Windows: venv\Scripts\activate
  7. On Windows PowerShell: venv\Scripts\Activate.ps1
  8. Your terminal prompt should now show the name of the virtual environment, indicating that it is active.

Here are some additional tips

  • To install packages, use pip install package_name.
  • To deactivate the virtual environment, simply run the command: deactivate.
  • To remove the virtual environment, delete the venv directory.
  • To ensure your virtual environment is not included in version control, add venv/ to your .gitignore file.
  • To recreate the virtual environment later, you can run the same python3 -m venv venv command again in your project directory.
  • To check if the virtual environment is active, you can run which python (macOS/Linux) or where python (Windows) to see the path of the Python interpreter being used. It should point to the venv directory.
